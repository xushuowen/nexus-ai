|----------|----------------|-----|
| `dashboard-main.png` | Dashboard 主控台 tab — show left stats panel + Nexus Core SVG + Daily Brief | `localhost:8000/dashboard` → 主控台 tab |

## Tips

- Resolution: **1920×1080** minimum
- Crop to show the most important content
- Make sure the ONLINE badge is green in all screenshots
- For `dashboard-chat.png`: run the ACL paper query first so results are visible
- For `skill-graph.png`: wait 3-5 seconds for D3.js simulation to settle
